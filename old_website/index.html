Projects

Machine Learning

Fairlearn: Responsible AI

This socio-technical work caters to enhancements in fairness constraint definition by generalising them and adding new disparity metrics. Also, experimentation with different techniques were performed to improve the time to train a classifier under fairness constraints. All contributions were directly incorporated in open source project named fairlearn, started by Microsoft.

Comparative study of BERT models for Open QA using CORD-19

This work addresses the task of extracting the relevant documents from the recent CORD-19. Implemented encoder-mapper based model: encoder - to create clusters from the contextual embeddings using BERTbase, SciBERT and BioBERT; mapper - to fine the most relevant cluster to answer a user provided question based on coronavirus; and then, compare/analyse the results from each type of BERT variant.

Image Captioning Model using Neural Nets

This work is an experiment towards image description and understanding the context of images by leveraging neural networks. Inspired from 'Show, Attend and Tell: Neural Image Caption Generation with Visual Attention', implemented encoder-decoder model to caption the image with pre-trained VGG19 and LSTM as baseline and then, improved the BLEU-4 score from 0.028 to 0.313 by incorporating ResNeXt and BERT as encoder and decoder respectively on MS-COCO.

Sentiment Analysis

On Tweets: This work is an effort to classify sentiments of a small dataset of the tweets scraped from Twitter, using various models like LogisticRegression, Naive Bayes, Decision Trees. Also, tried to improve the performance by incorporating various pre-processing techniques like Stemming, Lemmatization, cleaning, etc. and contextual embeddings derived from ELMO.
On Amazon Reviews: Built an implementation to classify amazon reviews in a public dataset from scratch using Naive Bayes after pre-processing the data.

WoMan: Wonderful Mangement

To get my hands dirty with chatbots and ML, this hobby project is a chatbot + webApp designed to manage files/spoken texts - converting speech-to-text, recording, indexing, tagging and fetching files, using a chatbot and visualise the connection between them using React-Force-Graph. Used pagerank algorithm for ranking the texts and spaCy for parsing text. Google Cloud Platform is used for the chatbot and database storage.

Design of genetic programming algorithm with simultaneous feature selection & its implementation for handling big data

This work is an attempt to implement an algorithm for multi-objective GP classifier, based on dynamic initialisation & crossover based on hill climbing approach, inspired from 'A novel genetic programming approach for epileptic seizure detection'. Also, worked on its scalability & simulated it for big datasets on MS Azure Spark clusters.
